# CanvasFlow

CanvasFlow, kullanıcıların çeşitli web içeriklerini ve araç takımlarını organize etmelerine olanak tanıyan dinamik bir dijital tuval uygulamasıdır.

## Geliştirme

Bu proje artık Firebase'den bağımsız olarak geliştirilmektedir. Veriler yerel depolama (localStorage/IndexedDB) üzerinde tutulmaktadır.

### Başlangıç

1. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```

2. Geliştirme sunucusunu başlatın:
   ```bash
   npm run dev
   ```

3. Tarayıcınızda [http://localhost:3000](http://localhost:3000) adresini açın.
